# <a name="manage-log-analytics-using-azure-resource-manager-templates"></a>Správa protokolu analýzy pomocí šablony správce prostředků Azure

Můžete použít [správce prostředků Azure šablony] (... / azure-resource-manager/resource-group-authoring-templates.md) k vytvoření a konfigurace pracovních prostorů pro analýzy protokolu. Příklady úkoly, které lze provést pomocí šablony:

+ Vytvoření pracovního prostoru aplikace Groove
+ Přidání řešení
+ Vytvoření uložená hledání
+ Vytvoření skupiny pro počítače
+ Povolení kolekce protokoly služby IIS z počítače s Windows agent nainstalovaný
+ Shromáždit výkonnosti z počítače Linux a Windows
+ Shromažďovat události z syslog Linux počítačů 
+ Shromažďovat události z protokoly událostí systému Windows
+ Shromáždění protokolů vlastní události
+ Přidání agent analýzy protokolu Azure virtuálního počítače
+ Konfigurace protokolu analýzy indexovat data shromažďované pomocí Azure diagnostiky


Tento článek obsahuje šablony ukázek, které ilustrují některé konfigurace, je možné provést ze šablony.

## <a name="create-and-configure-a-log-analytics-workspace"></a>Vytváření a konfigurace pracovní prostor protokolu analýzy

V následujícím příkladu šablony ukazuje postup:

1.  Vytvoření pracovního prostoru aplikace Groove
2.  Přidání řešení do pracovního prostoru
3.  Vytvoření uložená hledání
4.  Vytvoření skupiny pro počítače
5.  Povolení kolekce protokoly služby IIS z počítače s Windows agent nainstalovaný
6.  Shromáždit logické čítače výkonu z počítače Linux (% používá Inodes; Bezplatné megabajtů; % Použít místo; Přenosy disku/s; Čtení z disku/s; Zápisy na disk/s)
7.  Shromáždit syslog události z Linux počítačů
8.  Shromáždit chyb a upozornění události z protokolu událostí z počítače se systémem Windows
9.  Shromáždit dostupné MB paměti čítače z počítače se systémem Windows
10. Shromáždit vlastní protokol 
11. Shromáždit protokoly služby IIS a protokoly událostí Windows Azure diagnostiky který napsal k účtu úložiště

### <a name="deploying-the-sample-template"></a>Ukázková šablona nasazení

Ukázková šablona nasazení:

1. Uložení připojených vzorku v souboru, například`azuredeploy.json` 
2. Upravit šablonu, abyste měli požadovaná konfigurace
3. Pomocí prostředí PowerShell nebo příkazového řádku pro nasazení šablony

#### <a name="powershell"></a>Prostředí PowerShell

`New-AzureRmResourceGroupDeployment -Name <deployment-name> -ResourceGroupName <resource-group-name> -TemplateFile azuredeploy.json`

#### <a name="command-line"></a>Přepínač příkazového řádku

```
azure config mode arm
azure group deployment create <my-resource-group> <my-deployment-name> --TemplateFile azuredeploy.json
```
